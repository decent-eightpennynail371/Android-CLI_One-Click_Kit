# 왕초보 시작 가이드 (정말 쉽게)

> 컴퓨터·IT가 처음이어도 괜찮아요. 이 순서만 따라 하세요.

## 1. 이게 뭐예요?

구글이 만든 **Android CLI**(안드로이드 앱 개발 도구)를, 명령어 없이 **더블클릭만으로**
설치하고 쓰고 지울 수 있게 만든 도우미입니다.

## 2. 어떤 순서로 누르나요?

폴더 안의 파일을 **위에서 아래로** 더블클릭하면 됩니다.

1. **`시작하기.bat`** — 처음이면 여기부터. 안내 + 바로 실행 메뉴(설치/사용/제거/점검)를 띄워 줍니다. (이 파일 자체는 아무것도 안 깔립니다.)
2. **`자가진단.bat`** — (선택) 내 컴퓨터가 준비됐는지 점검. (아무것도 안 깔림)
3. **`INSTALL.bat`** — 설치하기.
4. **`RUN.bat`** — 메뉴를 열어서 사용하기.
5. **`UNINSTALL.bat`** — 다 쓰면 깨끗이 지우기.

> 잘 모르겠으면 `시작하기.bat` 부터 더블클릭하세요. 길을 안내해 줍니다.

## 3. 파란 경고창이 떴어요! (제일 많이 막히는 곳)

`"Windows의 PC 보호"` 같은 **파란 창**이 뜰 수 있어요. 이건 고장이 아닙니다.

1. **"추가 정보"** 글씨를 클릭
2. 아래에 생기는 **"실행"** 버튼을 클릭

→ 구글 공식 도구라서 안전합니다.

## 4. "관리자 권한으로 실행" 해야 하나요?

**아니요.** 그냥 **더블클릭**만 하세요. (관리자 권한은 필요 없습니다.)

## 5. 설치가 멈춘 것 같아요

설치할 때 **다운로드에 1~2분** 걸릴 수 있어요.
글자가 잠깐 안 움직여도 **창을 닫지 말고** 기다려 주세요.

## 6. 메뉴(RUN.bat)가 어려워요

- 번호를 **숫자로 입력**하고 **Enter** 를 누르면 됩니다.
- 처음이면 **`9` (쉬운 설명)** → 그다음 **`1` (상태 확인)** 을 눌러보세요.
- 안심하세요: 이 메뉴엔 무언가 **지우는 기능이 없어요**. (지우기는 `UNINSTALL.bat` 이 따로 `yes` 를 물어본 뒤에만 실행합니다.)

## 7. 지우고 싶어요

`UNINSTALL.bat` 더블클릭 → 화면이 물어보면 **`yes`** 입력.
(설치할 때 만든 것만 지우고, 다른 자료는 건드리지 않습니다.)

## 8. 막혔을 때

- `lib` 폴더는 **지우지 마세요** (키트가 작동 안 합니다).
- 글자가 **네모(ㅁ)·물음표(?)** 로 깨져 보이면 알려주세요.
- 무엇이든 막히면 그 **화면을 캡처**해서 도움을 요청하세요.
